На Главную Complex Vue form

Комплексная форма на Vue.js

В верстке проекта реализованы табы и аккордеон.

Использованные технологии и плагины:

  • Vue.js (Vue CLI, Router, Vuex)
  • Ckeditor.js — Rich-text editor
  • Simple-Vue-Validator —
  • V-chips-multiselect — select с множественным выбором
  • Генерация и подключение веб-шрифтов
  • Dom-to-image (получение изображения из HTML элемента)
  • Загрузка изображения на сервер (PHP)
  • Vue-social-sharing (кнопки Поделиться)
