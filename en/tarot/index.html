На Главную Daria-bloom.ru

Landing Page Divination by Tarot Cards (Pug, Scss, JavaScript, MODx)

Page sections:

  • Intro: short description of the service.
  • About: text and certificate.
  • Service: detailed description of the service.
  • Reviews: carousel.
  • FAQ: accordion with input type radio.
  • Card of the day: withdrawal of a random tarot card by clicking on the Select card button with the ability to do this only once a day (localStorage).
  • Gallery of images.
  • Contact with feedback form (PHP + JavaScript).

Completed work:

  1. Design: creation in Photoshop, drawing graphics, backgrounds and favicons.
  2. The Layout was carried out using the Pug and SASS template engines. The project was built using the Webpack bundler. Implemented: preloader, mobile menu, different images for different screens (<picture>), slider, accordion, gallery, modal windows.
  3. Integration with cms MODx. Using MIGX for FAQ, Testimonials and Photo Gallery blocks.

JS libraries used:

  • Slider Splide for reviews.
  • Lightgallery for images gallery.
