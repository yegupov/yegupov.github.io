Stopwatches

Task:

Implement a stopwatch widget. Buttons "stop", "start" (the same button), "reset", "circle". The stopwatch works with millisecond precision. When it is started, the time is displayed in the format hh: mm: ss: ms. Pressing the stop button stops the stopwatch.

The "circle" button adds the current stopwatch value to the list of results. Items from the list of results can be removed individually.

The "reset" button stops the stopwatch if it is running, resets the value to zero, clears all the results, if any. The "lap" works the same for stopped and started stopwatches.
