Chat application in Vue.js, Socket.IO, Nuxt.js, Vuetify.js.

Fullstack real-time chat using technologies:

  • Vue.js,
  • Nuxt.js,
  • Node.js (Express),
  • Socket.IO,
  • Vue-Socket.IO,
  • Vuetify.js.

The application does not use a database. Implemented text messages.
Hosted project on Heroku.
