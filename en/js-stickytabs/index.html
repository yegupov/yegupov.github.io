Responsive Sticky Menu Tabs Plugin (JavaScript, Html, Css)

Tabs with floating (sticky) menu in the left column (sidebar). Such an interface block will allow you to place large amounts of text and images, dividing them by topic, and to have the navigation menu by tabs constantly visible when you scroll the page.

Below on this page, see a demo of 2 blocks with such tabs.

Plugin description

Sticky Menu Tabs have specific Html markup that needs to be inserted into the page code. Each block will need to be assigned a unique class to activate the plugin.

Tabs consist of a sidebar (left column) that contains a sticky menu, and a content right column with the contents of the tabs. When scrolling down the page, the menu on the left is fixed at the top of the screen, thereby remaining in the user's field of view all the time. This makes the web page convenient to use. If you want to view another tab, you don't have to go back to the top to switch to it.

The Sticky Menu Tabs plugin implements:

  • adaptability: for screens less than 600 pixels wide, the content part expands to its full width, the sidebar menu is hidden, and a Menu button is added, when you click on it, you can expand and collapse all menu items;
  • expanding tabs: it is possible to expand all tabs on small screens and arrange them in a column;
  • resize tabs: when resizing the screen, the appearance of the tabs takes on an adequate predefined design;
  • image loading check: to accurately determine the height of the container for the floating block (sidebar menu) for the correct operation of sticky positioning technology, the loading of the document and all images of the first tab are checked before activating the plugin;
  • page scrolling to the top of the tabbed block: after clicking on any of the menu items, smooth scrolling to the top of the target tab is performed to view it. Thanks to this mechanism and the fixed menu on the left side of the screen, you do not need to scroll up the page to switch to other tabs.

HTML

<div class="sticky-tabs my-stickytabs">
  <div class="floating sticky-tabs__aside">
    <ul class="tabs">
      <li class="tab">Tab 1</li>
      <li class="tab">Tab 2</li>
      <li class="tab">Tab 3</li>
    </ul>
  </div>
  <div class="tab-content sticky-tabs__article">
    <div class="tab-item">
      <h3>Tab 1 Content</h3>
      <p>Some text and images.</p>
    </div>
    <div class="tab-item">
      <h3>Tab 2 Content</h3>
      <p>Some text and images.</p>
    </div>
    <div class="tab-item">
      <h3>Tab 3 Content</h3>
      <p>Some text and images.</p>
    </div>
  </div>
</div>

JavaScript code of Sticky Menu Tabs plugin

During the development, the prototype inheritance mechanism was used. The plugin JavaScript code is a constructor function with its properties and methods defined in the prototype property of this function.

Instantiation

var firstStickyTabsNode = document.querySelector('.first-stickytabs'),
    secondStickyTabsNode = document.querySelector('.second-stickytabs');

firstStickyTabsNode.stabs = new StickyMenuTabs({
  tabsNode: firstStickyTabsNode,
  menuName: 'Страницы сайта'
});

secondStickyTabsNode.stabs = new StickyMenuTabs({
  tabsNode: secondStickyTabsNode,
  menuName: 'Список услуг',
  mobileTabsOpen: true
});

CSS

The styles define: 2-column structure, indents, borders in the menu, sticky positioning for the menu block, the appearance of the Menu button and the menu itself for small screens. Some media styles are set.

Сюда можно отнести смену цветового решения сайта, добавление новых картинок, видео, анимаций. Таким образом, сам дизайн остается практически прежним, но более обновленным.
