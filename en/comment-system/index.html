Commenting system in PHP, MySQL, jQuery и AJAX

Reviews on the doctor's personal website COSMETOLOG-DRONOV.ru.

The project has implemented:

  1. Nesting comments with the ability to respond to reviews.
  2. Validate form fields on both the client and server side.
  3. Checking the comment text for links and "http".
  4. Inserting messages into MySQL database.
  5. Pagination: ajax comment loading.
  6. Zoom images.
  7. Image validation (<input type = "file">) on upload and on server: size and format.
  8. Show a preview of the image when loading (FileReader) in the form of adding a comment.
  9. Rating by star rating type.
  10. Styling the Administrator record.
  11. Displaying the form immediately below the Reply button and fixing the view at the same level after adding a comment.
  12. A notification letter to the Administrator about a new review.
  13. Spam protection with an additional field.
