Creating a REST API in PHP for the Site Admin

The project is designed to interact with the application of the Admin panel of the doctor's personal website COSMETOLOG-DRONOV.ru.

The REST API is written in PHP. Admin Panel Application - in Vue.js.

How the API interacts with the Admin Panel:

  1. During Authorization, the data entered in the Login form from the application goes to this-api.ru/login.php
  2. The server checks the entered data (login and password), generates a JWT token and sends it to the client.
  3. The client should now reset its token when transmitting any requests.
  4. The server, accepting the request to receive data, validates the received token and allows or denies access.

Highlights and what is developed:

  • Connection to the database and operations with it occur using PDO technology.
  • Creation, encoding and decoding of JWT (JSON Web Token) is done using the PHP-JWT library.
  • The project implements the operations of creating, reading, updating and deleting (CRUD) users and comments. The About Me and Contacts content blocks are read-only and update-only.
  • For Users, added the ability to upload images (avatars) to the server, validate (size and format) and delete old ones when changing avatars.
