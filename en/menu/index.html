Context menu

Task:

Implement ContextMenu.

The first argument is the node on which the context menu will run.

The second argument is a description of the menu structure. An example description is attached. The structure can be any in terms of nesting and the number of menu items (recursively generating the DOM will be ok).

A submenu can contain nested submenus.

When right-clicking on the node for which the ContextMenu was created, show the menu right below the clicked point.

When clicking on a menu item, the corresponding function should be performed.

Each item containing a submenu must be marked with a ">" symbol.

The submenu opens when you hover over a submenu item, and closes when the mouse leaves the submenu or submenu item (see mouseenter and mouseleave events).

You don't have to bother with positioning (so that all menus and submenus fit into the visible part of the screen).

You can style the menu to your liking (the main thing is the visible borders of the elements)

An example of a menu structure object:

        
  var menuExample = [{
      title: File,
      action: function () {console.log(open file)}
  }, {
      title: Edit,
      action: function () {console.log(edit content)}
  }, {
      title: More stuff,
      submenu: [{
          title: Send by email,
          action: function () {console.log(emailed)}
      }, {
          title: Send via skype,
          action: function () {console.log(skyped)}
      }]
  }]
