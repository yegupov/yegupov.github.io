Draft training course on Loftschool.com (10/07/2019 - 11/13/2019)

The project has developed a one-page website for a design layout and an administrative panel (SPA).

Website

View page

  1. Home screen - parallax effect with image layers and JavaScript.
  2. Skills block - Vue component, SVG + JavaScript
  3. Slider My Works - Vue component
  4. Reviews Slider - Via Flickity Plugin
  5. Feedback form - validation for filling in fields using the Simple Vue Validator plugin

Admin panel

Login page

The project admin panel is implemented as a single page web application written using Vue.js.

  • Authorization
  • Working with API
  • Routing
  • Creating, editing and deleting items
  • Validation of fields and messages for the user
  • Displaying data from the database on the site

Technology and skills

  • Responsive layout (flexbox, grid, img srcset)
  • BEM
  • SVG
  • Parallax effect (multiple images + JavaScript)
  • JavaScript (ES6)
  • Vue.js (widgets, single file components, plugins, animations, router, vuex, axios)
  • Workflow (Pug, Pcss, Webpack, Git, Terminal)
