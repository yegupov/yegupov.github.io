Secret Santa landing page

«Become Secret Santa for disabled children» – landing page with responsive design.

The layout was carried out using the Pug and SASS template engines. Bootstrap grid (reboot + grid) was applied. The project was built using the Webpack bundler.

What is on the site:

  • Sliders (Slick).
  • Pop-up windows (Fancybox) with form.
  • Text in a semicircle in CSS and SVG (phrase: To which of this will the children say «wow»?).
