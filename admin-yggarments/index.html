Админ-панель интернет-магазина на Vue.js

Приложение (SPA) создано для PHP магазина YG-GARMENTS.ru. В нем присутствуют следующие страницы: Пульт (Главная), Товары, Категории, Коллекции, Блог и Пользователи.

Для осуществления операций создания, чтения, обновления и удаления данных (CRUD) разработано REST API (PHP).

С целью создания SVG спрайта был заменен базовый загрузчик Vue CLI на svg-sprite-loader путем настройки файла vue.config.js.

Посмотреть страницу авторизации приложения.

Функционал, реализованный в приложении:

  1. Авторизация и выход из Админ-панели.
  2. Валидация полей форм при помощи плагина Simple-Vue-Validator и отображение подсказок.
  3. Создание, чтение, обновление и удаление (CRUD) товаров, категорий, коллекций, постов блога и пользователей.
  4. Чтение сделанных в магазине заказов и отбражение их в виде графиков и диаграмм (Highcharts) с фильтрацией по количеству, общей сумме и годам (страница Пульт).
  5. Загрузка изображений на сервер и удаление прежнего при смене или удалении администратором.
  6. Пагинация в разделах Товары и Блог.
  7. Выбор количества позиций для показа на одной странице.
  8. Фильтрация товаров по разделам каталога, категориям и коллекциям. Фильтрация постов блога по годам.
  9. Сортировки товаров по id и цене.
  10. Поиск по названию товаров и постов.
  11. Сообщения-подсказки для администратора в виде всплывающих внизу модальных окон.
  12. Для главной страницы Пульт разработано отображение статистики из Яндекс Метрики. Использованы API Яндекс Метрики и библиотека Highcharts для построения графиков. Также на странице выводятся актуальные курсы валют и имеется калькулятор.

Использованные инструменты и пакеты:

  • Vue CLI — система для быстрой разработки на Vue.js.
  • Шаблонизатор Pug.
  • Препроцессор Sass.
  • Для Svg спрайта использованы пакеты: svg-sprite-loader, svgo и svgo-loader.
  • Axios — для получения и отображения данных из API.
  • Vuex — для работы с данными (управление состоянием).
  • Highcharts и highcharts-vue — для построения диаграмм и графиков.
  • Vue2-editor — текстовый редактор.
