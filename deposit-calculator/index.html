Депозитный калькулятор

Калькуляторы вкладов предназначены для расчета доходности по банковским вкладам на различные сроки.

Описание

Проект разработан при помощи Javascript и Vue.js. На странице отображаются два варианта (плана) вклада. Все числовые значения и названия планов формируются генератором случайных данных Faker.
