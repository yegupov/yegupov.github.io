На Главную Посмотреть лендинг

Проект учебного курса на Loftschool.com (07.10.2019 – 13.11.2019)

В проекте разработаны одностраничный сайт по дизайн-макету и админпанель (SPA).

Сайт

Посмотреть страницу

  1. Главный экран – параллакс эффект с помощью слоев с картинками и JavaScript.
  2. Блок Навыки – компонент Vue, SVG + JavaScript
  3. Слайдер Мои работы – компонент Vue
  4. Слайдер Отзывы – с помощью плагина Flickity
  5. Форма обратной связи – валидация на заполнение полей  при помощи плагина Simple Vue Validator

Админпанель

Страница авторизации

Админпанель проекта реализована в виде одностраничного веб-приложения, написанного с использованием Vue.js.

  • Авторизация
  • Работа с API
  • Роутинг
  • Создание, редактирование и удаление элементов
  • Валидация полей и сообщения для пользователя
  • Отображение данных из БД на сайте 

Технологии и навыки

  • Адаптивная верстка (flexbox, grid, img srcset)
  • БЭМ
  • SVG
  • Параллакс эффект (несколько картинок + JavaScript)
  • JavaScript (ES6)
  • Vue.js (виджеты, однофайловые компоненты, плагины, анимация, router, vuex, axios)
  • Workflow (Pug, Pcss, Webpack, Git, Terminal)
